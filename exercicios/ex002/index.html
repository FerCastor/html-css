Parágrafos e quebras de linha

Este é o teste do primeiro paragrafo.

Para quebrar o texto é só usar a tag <br> antes da palavra que deseja que seja a primeira da próxima linha.

Para escrever um parágrafo, basta colocar o texto desejado entre as tags: <p> Texto de exemplo </p>
